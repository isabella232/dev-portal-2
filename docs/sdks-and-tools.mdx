The following SDKs and tools are actively developed and supported by Vertex. They provide components and tools which simplify and accelerate the development of applications leveraging the Vertex platform. These tools receive regular product updates and patches.

| Tool                        | Repository                                                                              | Package Registry                                                                                                                                             | Documentation                                                |
| :-------------------------- | :-------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------- |
| Vertex CLI                  | [GitHub](https://github.com/Vertexvis/vertex-cli)                                       | [![npm (scoped)](https://img.shields.io/npm/v/@vertexvis/vertex-cli?style=for-the-badge)](https://www.npmjs.com/package/@vertexvis/vertex-cli)               | [TypeDoc](https://vertexvis.github.io/vertex-cli/)           |
| Web UI Component            | [GitHub](https://github.com/Vertexvis/vertex-web-sdk)                                   | [![npm (scoped)](https://img.shields.io/npm/v/@vertexvis/viewer?style=for-the-badge)](https://www.npmjs.com/package/@vertexvis/viewer)                       | [TypeDoc](https://vertexvis.github.io/vertex-web-sdk/)       |
| React UI Component bindings | [GitHub](https://github.com/Vertexvis/vertex-web-sdk/tree/master/packages/viewer-react) | [![npm (scoped)](https://img.shields.io/npm/v/@vertexvis/viewer-react?style=for-the-badge)](https://www.npmjs.com/package/@vertexvis/viewer-react)           |                                                              |
| API Client for Node.js\*    | [GitHub](https://github.com/Vertexvis/vertex-api-client-ts)                             | [![npm (scoped)](https://img.shields.io/npm/v/@vertexvis/vertex-api-client?style=for-the-badge)](https://www.npmjs.com/package/@vertexvis/vertex-api-client) | [TypeDoc](https://vertexvis.github.io/vertex-api-client-ts/) |
| API Client for Java\*       | [GitHub](https://github.com/Vertexvis/vertex-api-client-java)                           |                                                                                                                                                              |                                                              |
| Xamarin UI Components       |                                                                                         | [![NuGet](https://img.shields.io/nuget/v/Vertexvis.Viewer?style=for-the-badge)](https://www.nuget.org/packages?q=Vertexvis)                                  |                                                              |

\* If you use a different language, either [let us know](/contact), use the [`openapi-generator`](https://github.com/OpenAPITools/openapi-generator) to generate a client, or follow the [API Reference](https://docs.vertexvis.com/) and use the framework of your choice.
